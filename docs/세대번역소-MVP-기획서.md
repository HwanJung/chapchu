# 세대번역소 MVP 기획서

## 1. 제품 개요

### 1.1 서비스명

**세대번역소** (가칭)

### 1.2 기획 배경

세대마다 익숙한 단어와 말투가 달라 같은 내용도 다르게 이해되거나 의도하지 않은 갈등으로 이어질 수 있다. 세대번역소는 이러한 언어 차이를 문맥에 맞게 번역하고, 상대방이 표현을 어떻게 받아들일 수 있는지 함께 설명하는 서비스다.

### 1.3 서비스 목표

- MZ세대와 중장년·노년층 사이의 언어 차이로 발생하는 오해를 줄인다.
- 단순한 단어 치환을 넘어 문맥과 격식에 맞는 자연스러운 문장을 제공한다.
- 낯선 표현의 뜻과 뉘앙스를 설명해 세대 간 언어 이해를 돕는다.
- 교수님, 직장 동료, 친구 등 상대방과의 관계에 적합한 문장 작성을 지원한다.

### 1.4 주요 사용자

- 신조어와 줄임말의 뜻을 알고 싶은 중장년·노년층
- 어르신의 표현이나 격식 있는 문장을 이해하고 싶은 청년층
- 교수님이나 직장 관계자에게 보낼 문장을 다듬고 싶은 사용자
- 가족이나 지인과의 세대 차이로 의사소통에 어려움을 느끼는 사용자

### 1.5 핵심 가치

> 단어만 바꾸는 번역기가 아니라, 표현에 담긴 의미와 상대가 받는 느낌까지 전달한다.

---

## 2. 서비스 구성

서비스는 다음 두 개의 탭으로 구성한다.

1. **세대어 번역**: MZ어와 어르신어를 양방향으로 번역한다.
2. **상황별 말 바꾸기**: 입력한 문장을 상대와의 관계에 맞게 변형한다.

---

## 3. 핵심 기능

### 3.1 세대어 번역

#### 번역 방향

- MZ어 → 어르신어
- 어르신어 → MZ어
- 전환 버튼으로 번역 방향 변경

#### 문장 입력

- 원문 입력 영역 제공
- 최대 500자 입력
- 입력 내용 전체 삭제 기능
- 입력 글자 수 표시

#### 격식 강도 조절

번역 결과의 격식을 다음 5단계로 조절한다.

| 단계 | 명칭 | 표현 예시 |
|---|---|---|
| 1 | 매우 정중함 | 안녕하십니까 |
| 2 | 정중함 | 안녕하세요 |
| 3 | 보통 | 안녕 |
| 4 | 친근함 | ㅎㅇ |
| 5 | 매우 캐주얼 | ㅎ2 |

- 기본값은 3단계로 설정한다.
- 슬라이더 조작이 끝난 시점에 선택한 단계의 결과를 다시 생성한다.
- 예시는 강도의 이해를 돕기 위한 것으로, 실제 결과는 문맥과 번역 방향에 따라 달라진다.

#### 번역 결과

- 선택한 번역 방향과 격식 단계에 맞는 결과 한 개를 크게 표시한다.
- 결과 복사 기능을 제공한다.
- 번역에 실패하면 오류 안내와 재시도 버튼을 제공한다.

### 3.2 표현 설명

번역 결과 하단에 문장에서 사용된 주요 세대 표현을 사전 형태로 설명한다.

표현별로 다음 정보를 제공한다.

- 표현 또는 단어
- 기본적인 뜻
- 상대방에게 줄 수 있는 느낌과 뉘앙스
- 주로 사용하는 상황
- 오해하거나 무례하게 느낄 수 있는 상황

한 번역 결과에서 최대 5개 표현을 안내하며, 설명할 표현이 없으면 설명 영역을 표시하지 않는다. 결과 문장의 해당 표현과 설명 카드를 시각적으로 연결한다.

### 3.3 상황별 말 바꾸기

사용자가 전달하려는 내용을 입력하면 상대방과의 관계에 적합한 문장으로 변형한다.

#### 대상 선택

MVP에서는 다음 6개 대상을 제공한다.

- 교수님
- 직장 상사
- 회사 동료
- 친구
- 연인
- 가족

#### 변형 원칙

- 상대방에게 적합한 호칭, 인사, 어휘와 문장 구조를 적용한다.
- 필요한 경우 완곡한 표현과 자연스러운 맺음말을 보완한다.
- 원문의 핵심 의도와 사실관계는 유지한다.
- 원문에 없는 일정, 장소, 사유, 약속 등 구체적인 사실은 생성하지 않는다.
- 관계별 결과는 가장 적합한 기본 말투 한 가지로 제공한다.

#### 결과 기능

- 변형된 문장 표시
- 결과 복사
- 오류 발생 시 재시도

## 4. 화면 구성

### 4.1 공통 영역

- 서비스 로고 및 이름
- `세대어 번역`, `상황별 말 바꾸기` 탭
- 서비스 이용 안내
- AI 생성 결과 및 세대 표현의 개인차에 관한 안내

### 4.2 세대어 번역 화면

1. 번역 방향 선택 영역
2. 원문 입력 영역
3. 격식 강도 슬라이더
4. 번역 실행 버튼
5. 번역 결과 및 복사 버튼
6. 주요 표현 설명 카드

### 4.3 상황별 말 바꾸기 화면

1. 전달할 내용 입력 영역
2. 상대방 선택 버튼
3. 문장 변형 실행 버튼
4. 변형 결과 및 복사 버튼

### 4.4 공통 상태

- 입력 전 빈 화면
- AI 결과 생성 중
- 결과 생성 완료
- 서버 또는 AI 오류
- 입력 길이 초과
- 안전 정책상 처리할 수 없는 요청

## 5. 시스템 인터페이스

### 5.1 세대어 번역 API

#### 요청

```json
{
  "inputText": "점메추 부탁",
  "direction": "MZ_TO_SENIOR"
}
```

#### 응답

```json
{
  "resultText": "점심 메뉴를 추천해 주세요.",
  "termExplanations": [
    {
      "term": "점메추",
      "meaning": "점심 메뉴 추천의 줄임말",
      "nuance": "친근하고 가벼운 느낌을 주는 온라인 표현",
      "usage": "친구나 가까운 동료와 메뉴를 정할 때 사용",
      "caution": "격식 있는 자리에서는 뜻이 전달되지 않을 수 있음"
    }
  ]
}
```

### 5.2 상황별 말 바꾸기 API

#### 요청

```json
{
  "inputText": "과제 제출을 하루 늦게 해도 될까요?",
  "audience": "PROFESSOR"
}
```

#### 응답

```json
{
  "resultText": "안녕하세요, 교수님. 부득이하게 과제 제출을 하루 늦춰도 괜찮을지 여쭙고 싶습니다. 확인해 주시면 감사하겠습니다."
}
```

### 5.3 주요 데이터 타입

```text
direction:
  MZ_TO_SENIOR | SENIOR_TO_MZ

audience:
  PROFESSOR | BOSS | COWORKER | FRIEND | PARTNER | FAMILY

termExplanation:
  term | meaning | nuance | usage | caution
```

### 5.4 AI 처리 원칙

- AI는 정해진 JSON 구조로 결과를 반환한다.
- 서버는 필수 항목, 글자 수와 허용된 선택값을 검증한다.
- 동일한 입력, 방향과 격식 단계의 반복 요청은 단기 캐시할 수 있다.
- 번역 결과와 표현 설명에는 `AI 생성`임을 표시한다.
- 세대 표현은 개인과 상황에 따라 다를 수 있음을 함께 안내한다.

---

## 6. 개인정보 및 운영 정책

- 회원가입 없이 사용할 수 있다.
- 사용자가 입력한 원문과 생성 결과는 영구 저장하지 않는다.
- 서비스 품질 측정을 위해 번역 완료, 결과 복사, 오류, 응답 시간 등의 익명 이벤트만 수집한다.
- 익명 이벤트에는 입력 문장과 생성 결과를 포함하지 않는다.
- 세대별 말투를 고정관념이나 유일한 정답으로 단정하지 않는다.

---

## 7. 성공 지표

- 번역 실행 대비 정상 결과 생성 비율
- 결과 복사율
- 동일 문장의 반복 생성 비율
- API 오류율
- 평균 및 상위 95% 응답 시간
- 사용자 검수에서의 의미 보존율과 표현 자연스러움

초기 품질 검수는 세대, 번역 방향, 격식 단계와 관계 유형별 대표 문장 세트를 구성해 진행한다. 청년층과 중장년·노년층 참여자가 결과의 자연스러움과 불쾌 가능성을 교차 평가한다.

## 8. MVP 제외 범위

다음 기능은 초기 버전에서 제외하고 후속 버전에서 검토한다.

- 회원가입 및 로그인
- 번역 기록과 즐겨찾기
- 사용자 평가 및 신고
- 직접 관계 입력
- 음성 입력 및 읽어주기
- 이미지 문자 인식(OCR)
- 다국어 번역
- 사용자 참여형 세대어 사전
- 개인별 말투 학습

---

## 9. MVP 완료 기준

- 세대어 양방향 번역이 정상 작동한다.
- 격식 5단계가 결과에 일관되게 반영된다.
- 주요 표현의 뜻과 뉘앙스가 결과 하단에 표시된다.
- 6개 관계에 대한 상황별 문장 변형이 가능하다.
- 상황별 변형 결과가 원문에 없는 구체적인 사실을 만들지 않는다.
- 모바일과 데스크톱에서 핵심 기능을 사용할 수 있다.
- 키보드만으로 입력, 대상 선택, 번역 및 결과 복사가 가능하다.
- 사용자의 원문과 결과가 영구 저장되지 않는다.
